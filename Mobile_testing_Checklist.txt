Гайдлайны IOS 
1. Приложение можно обновить только через магазин приложений. 
2. Приложение не имеет доступ к файлам, находящимся вне приложения, без разрешения пользователя. 
3. Приложение не имеет доступа к к папкам и файлам вне директорий контейнера и документов. 
4. В приложении не упоминаются сторонние платформы.
5. Приложение не использует местонахождение пользователя без разрешения. 
6. Функциональность соответствует описанию в магазине приложений. 
7. Приложение имеет уникальное имя из не более, чем 30 символов.
8. Приложения не разряжает быстро аккумулятор, не нагревает чрезмерно устройство или чрезмерно загружает ресурсы устройства. 
9. Приложение использует только общедоступные API.
10. Приложение полностью функционально в сетях, поддерживающих только IPv6.
11. Приложения, которые используют стороннюю службу или службу входа в социальные сети (например, вход в Facebook, вход в Google, вход в Twitter, вход в LinkedIn, вход в Amazon или вход в WeChat) также предлагает вход с помощью Apple в качестве эквивалентной опции.
12. Приложение не является "опытной", "бета", "демо" или "тест"-версией. 
13. Приложение не рассылает спам, не распространяет вирусы.
14. Приложение не может использовать уведомления без согласия пользователя.
15. Уведомления должны рассылаются через Apple Push Notification API, при помощи APN ID.  
16. Уведомления не содержат личных данных. 
17. Рекламные банеры можно скрыть.
18. Приложение не нарушает авторские права.
19. После загрузки приложение работает. Оно не должно отключается через несколько дней. 
20. Приложение не использует устаревшие интерфейсы.

Гайдлайны Android 
1. Приложением имеет название из не более, чем 30 символов.
2. Размер apk-файла не превышает более 100 Мб (и 50 Мб для Android 2.2 и ниже, или для Play Market 5.2 и ниже).
3. apk-файл должен подписан файлом цифровой подписи.
4. Все личные и конфиденциальные данные пользователей обрабатываются безопасным образом, в том числе с применением современных методов шифрования, например протокола HTTPS.
5. В приложении содержится информация о сборе, использовании и передаче личных данных.
6. Приложение не нарушают работу устройства пользователя, других устройств, компьютеров, серверов, сетей, API или сервисов, а также не вмешивается в их работу, не получает несанкционированный доступ к ним или вредят иным образом.
7. Все конфиденциальные данные хранятся во внутренней памяти приложения.
8. Приложение поддерживает биометрическую аутентификацию для защиты финансовых транзакций или конфиденциальной информации, такой как важные пользовательские документы.
9. Весь сетевой трафик отправляется через SSL.
10. Приложение поддерживает стандартную навигацию по кнопке «Назад».
11. Приложение поддерживает навигацию с помощью жестов для возврата / перехода на главный экран.
12. Когда приложение возобновляется из переключателя приложений «Недавние», оно возвращает пользователя в то состояние, в котором оно использовалось в последний раз.
13. Когда приложение возобновляется после того, как устройство выходит из спящего (заблокированного) состояния, оно возвращает пользователя в то состояние, в котором оно использовалось в последний раз.
14. Уведомления не используются для продвижения или рекламы другого продукта.
15. Приложение поддерживает как альбомную, так и портретную ориентацию.
16. Приложение правильно обрабатывает быстрые переходы между ориентациями дисплея и сворачиванием / разворачиванием без проблем с рендерингом или потери состояния.
17. Приложение отображает графику, текст, изображения и другие элементы пользовательского интерфейса без заметных искажений, размытия или пикселизации.
18. Приложение отображает текст и текстовые блоки корректно для каждого из поддерживаемых языков.
19. Если воспроизведение звука является основной функцией, приложение поддерживает фоновое воспроизведение.
20. Приложение не дает сбоев и не блокирует поток пользовательского интерфейса, вызывающий ошибки ANR.
